# SAIL Newsletter Web

This is the static web version generated from `2026.06.pdf`.

Open `index.html` directly in a browser, or publish the whole `web/` folder as a static site.

## Monthly update

1. Open `assets/content.js`.
2. Copy the whole `issues["2026-06"]` block and rename it, for example `issues["2026-09"]`.
3. Replace `monthLabel`, `cnMonth`, `sourcePdf`, `pdfPages`, Spotlight, Interview, Events, Resources, and Contact content.
4. Change `defaultIssue` to the new issue key.

Images go in `assets/images/`.
PDF preview images go in `assets/pdf-pages/`.
Original PDFs go in `assets/pdf/`.
